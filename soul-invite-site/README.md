# Soul Invite Site

Готовый React/Vite проект свадебного приглашения.

## Как запустить локально

```bash
npm install
npm run dev
```

## Как выложить по ссылке

### Netlify
1. Загрузите проект в GitHub.
2. Откройте Netlify.
3. Нажмите **Add new project**.
4. Подключите репозиторий.
5. Build command: `npm run build`
6. Publish directory: `dist`

### Vercel
1. Загрузите проект в GitHub.
2. Откройте Vercel.
3. Import Project.
4. Framework Preset: `Vite`
5. Нажмите Deploy.

## Что менять
- Основной контент: `src/App.jsx`
- Цвета и базовые стили: `src/styles.css`
